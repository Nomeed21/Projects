#Money Manager Interface

A lightweight, C-based command-line tool that helps users manage their finances using the **50/30/20 budgeting rule**. This project demonstrates file handling, dynamic data structures, and persistent storage in C.

## Features:
* **Automatic Allocation:** Automatically splits income into Needs (50%), Savings (30%), and Wants/Growth (20%).
* **Persistence:** Uses binary file handling to save financial data between sessions.
* **Transaction Logging:** Records expenses and income securely.
* **Cross-Platform:** Runs on Windows, Linux, and macOS.
